Microbrewery Dashboard

A dashboard that recieves updates from multipe sources (Websocket,SMS,Database update) that simulate the daily operations of a MicroBrewery. The dashboard is a Single Page Application that allows the user to easily view and filter any of these status update messages.

  • Single Page Application (SPA)
  • Vue,Vuex and Vue Router
  • Websockets
  • MicroService Architecture (Redis,Twilio,Slack)

Background: I wanted to put together a project that used all of the Vue technologies that I felt very comfortable with and create an attractive user interface.

Difficulties: First time using Vue Router but it was quite straight forward after looking in to it.

What I Learned: The strength of highly cohesive systems that utilize multiple web services to create strong applications but with a small codebase.
